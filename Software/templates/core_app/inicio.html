🔌 Consumo de API — Datos Abiertos Colombia

El framework se comunica con datos.gov.co mediante la API SODA oficial para consultar microdatos de educación y demografía en tiempo real.

# API Socrata (SODA) — Datos Abiertos Colombia
import requests, pandas as pd

API = "https://www.datos.gov.co/resource/upr9-nkiz.json"

def cargar_datos(limit=5000):
    r = requests.get(API, params={"$limit": limit})
    return pd.DataFrame(r.json()) if r.ok else pd.DataFrame()
💡 Sincronización: El pipeline ETL ejecuta este proceso asíncronamente, limpia e ingesta los datos directamente en la base de datos del modelo.
